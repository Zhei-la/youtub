시작 가이드
3분 안에 첫 쇼츠 기획을 만드는 방법
  1. API 키 등록 — 좌측 메뉴 [API 키]에서 OpenAI 키를 입력하세요. sk-로 시작합니다.
  2. 주제 입력 — [스튜디오]에서 시리즈 제목과 주제를 자유롭게 작성하세요.
  3. 스타일 선택 — 애니/실사/3D 등 원하는 스타일과 분위기를 선택합니다.
  4. 장면 생성 — [장면 4컷 생성] 버튼을 누르면 GPT가 후킹→전개→반전→엔딩 4컷을 만듭니다.
  5. 저장 / 다운로드 — JSON 또는 TXT로 내려받거나 히스토리에 저장합니다 (서버 저장).
  6. 이어쓰기 — 히스토리에서 이전 화를 불러오면 자동으로 [이어쓰기] 모드가 됩니다.
자주 묻는 질문

Q. 내 API 키는 안전한가요?
키는 브라우저 localStorage에만 저장되며, ShortsForge 서버로 절대 전송되지 않습니다. 모든 API 호출은 브라우저 → OpenAI 직통입니다.

Q. 비용은 누가 부담하나요?
OpenAI 호출 비용은 등록한 본인 계정으로 청구됩니다.

Q. 히스토리는 어디 저장되나요?
로그인 계정 기준으로 서버 DB에 저장되며, 기기를 바꿔도 동일하게 조회됩니다. 본인 계정의 데이터만 조회됩니다.
